Can users browse without an account?

Yes. Guest browsing remains available for the core map and search experience. Sign-in is required only for account-bound features such as favorites, submissions, moderation, and business claims.

How does account deletion work?

The app exposes both deactivation and permanent deletion from the profile surface. Deletion removes personal account data while anonymized community contributions may remain.

Where should stores and reviewers be sent?

Use this support page as the public support URL, the privacy page for privacy-policy metadata, and the terms page for public product rules. If reviewer credentials are required for a future gated web surface, provide them in store review notes.

What if a business dashboard needs backend analytics later?

The current browser dashboard is intentionally useful without fabricated metrics. It can accept real data later without changing the interaction model or public URL structure.
