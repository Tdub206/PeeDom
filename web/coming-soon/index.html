Coming soon

Future StallPass features we have already talked about.

This page collects the ideas currently on our roadmap from product conversations and planning notes. These features are not live yet.

Everything on this page is planned or discussed, not shipped
Includes near-term work and bigger future ideas

Next features we want to add

These are the most grounded features we have already discussed for StallPass, especially around business tools, access, trust, and daily operations.

Trust

Expanded StallPass Verified badges

Clearer badges for verified bathrooms and verified businesses, so guests can tell which locations have been confirmed.

Access

NFC lock support and tap-to-enter access

Support for partner restroom locks so participating locations can let approved guests tap with their phone instead of typing a code.

Business

Business photo uploads

Let owners upload restroom photos from the dashboard so listings feel more trustworthy and easier to review.

Live updates

Real-time notifications

Alerts for claim reviews, listing updates, code changes, featured campaigns, and other business activity as it happens.

Insights

Better analytics and trend charts

Deeper reporting with 7, 30, and 90 day trends so businesses can see what is improving and what needs attention.

Codes

Stronger access-code controls

Better tools for business-set codes, code rotation, lock settings, and clearer confidence signals around what still works.

Promotion

Featured placements and campaign tools

Self-serve ways for businesses to request more visibility in the app and track how those placements perform.

AI

AI help for business owners

An assistant that can help write updates, explain dashboard activity, and surface easy next steps inside the business console.

Security

Stronger account security

Extra protections like two-step sign-in and better account controls for businesses managing public listings.

Bigger ideas already on the table

These are the larger future ideas we have already discussed. They are longer-range than the items above, but they are still part of the broader StallPass vision.

Camera

AI code recognition

Take a photo of a keypad and let StallPass help read the code automatically.

Navigation

Indoor positioning

Better help inside malls, airports, and other large buildings where a bathroom is hard to find once you are already inside.

Wearables

Apple Watch companion

Quick nearby bathroom access and simple trust checks from a watch without reaching for your phone.

Widgets

iPhone widgets

Home screen or lock screen shortcuts for nearby bathrooms and useful real-time status.

Driving

CarPlay and Android Auto

Bathroom discovery while driving so road trips and delivery routes are easier to manage safely.

Confidence

Arrival-confidence score

A smarter score based on code recency, live status, hours, and trust signals to help people judge whether a stop is worth it.

Patterns

Time-based bathroom insights

Helpful notes like usually busy now or usually clean at this hour, based on real usage patterns over time.

Community

Contributor reputation

More weight for reliable contributors, plus better handling for false reports, bad codes, and repeat abuse.

Business scale

Enterprise tools for chains

Multi-location controls, higher-tier business tools, and support for brands managing many bathrooms at once.

Premium

Points that unlock premium

Let active contributors earn enough points to unlock premium access instead of only paying for it.

Progress

Contributor levels

A fuller level system so people can grow from new contributor to trusted expert over time.

What this page means

Some trust, verification, code, and business features already exist today. This page is for the expanded versions we still want to build next. As the roadmap changes, this list will change too.
