# PeeDom
Mobile Application to find Bathrooms

# 1. Initialize new Expo project
npx create-expo-app@latest peedom-mobile --template blank-typescript

# 2. Copy all files above into the project structure

# 3. Install dependencies
npm install

# 4. Install development client
npx expo install expo-dev-client

# 5. Create development build (Android)
npx expo run:android

# 6. Create development build (iOS)
npx expo run:ios

# 7. Start Metro bundler
npx expo start --dev-client
